# スキルの探し方 & インストール

自分で作らなくても、コミュニティが作ったスキルを利用できます。

---

## スキルを探せる場所

| プラットフォーム | 特徴 |
|---|---|
| **Anthropic公式マーケットプレイス** | Claude Code内の`/plugins`コマンドでアクセス |
| **Skills.sh** | インストール数ベースの人気ランキングを提供 |
| **Skills.mcp** | カテゴリ別整理で分野別探索に便利 |
| **Skill.bu** | ブラウザで試せるプレイグラウンド（準備中） |

---

## おすすめスキル

### Find Skills（はじめての方に必須）
スキルを探してくれるスキルです。自然言語でリクエストすると、関連スキルとインストールコマンドを案内してくれます。
```
"カードニュースを作るスキルを探して"
```

### Commit Command
コード修正後にコミットメッセージを自動作成するプラグインです。

### Skill Creator
オリジナルのカスタムスキルを作ってくれるツールです。→ [スキルを自作する](./skills.md) 参照

---

## インストール方法

Claude Code内の`/plugins`コマンドでマーケットプレイスにアクセスするか、ターミナルから直接インストールできます。

---

## ⚠️ 注意事項

- **インストール数の上限**: 多すぎるとAIが一部のスキルを認識できなくなることがあります。**40個前後**を目安にしてください。
- **セキュリティ確認**: 外部スキルにはファイル削除や外部呼び出しが含まれる場合があります。インストール前にGitHubでマークダウンファイルを確認してください。
