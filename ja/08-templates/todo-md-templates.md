# TODO.md テンプレート集

セッション間の作業継続性を維持するためのTODO.mdテンプレートです。コピー＆ペーストしてプロジェクトルートに`TODO.md`として保存してください。

---

## 基本テンプレート

```markdown
# [プロジェクト名] TODO

## 🔥 現在進行中
- [ ] 

## ✅ 完了
- [x] プロジェクト初期セットアップ (2025-00-00)

## 📋 次の予定
- [ ] 

## 🚫 ブロッカー（詰まっているもの）
- 

## 📝 意思決定メモ
<!-- なぜそう決定したかを日付とともに記録 -->
- 2025-00-00: 
```

---

```markdown

- [x] プロジェクト初期セットアップ

### 進行予定
- [ ] 

## 🐛 既知のバグ
- [ ]

- 

## 📝 意思決定メモ
- 2025-00-00: 
```

---

## CLAUDE.mdに記載するTODO読み取りルール

```markdown
## 作業ルール
- セッション開始時に必ずTODO.mdを読んで現在の状態を把握する
- 機能完了のたびにTODO.mdを更新する（完了項目にチェック、次の予定を追加）
- 重要な技術的決定は意思決定メモセクションに日付とともに記録する
- セッション終了前に今日完了した内容と次の作業をTODO.mdにまとめてからコミットする
```

---

## セッション別使用例

**セッション開始時のプロンプト:**
```
TODO.mdを読んで今日進める作業を把握した後、Planを立ててください。
```

**作業完了時のプロンプト:**
```
先ほど完了した[機能名]をTODO.mdにチェックして、次の予定項目も更新してください。
```

**セッション終了前のプロンプト:**
```
今日完了した作業、次に続ける作業、重要な決定事項を
TODO.mdにまとめてgit commitしてください。
```
