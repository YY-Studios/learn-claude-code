# 핵심 명령어 및 단축키 (Quick Reference)

클로드 코드의 효율적인 토큰 관리와 작업 속도를 200% 끌어올려 줄 필수 명령어 및 단축키 모음입니다.

## ⌨️ 꼭 알아야 할 핵심 단축키
* `Shift + Tab`: **플랜(Plan) 모드 ↔ 에딧(Edit) 모드 전환** (코드를 바로 수정하지 않고 계획부터 세울 때 필수)
* `ESC` 1번: 진행 중인 작업 즉시 중단 (Interrupt)
* `ESC` 2번 연속: **되감기(Rewind).** 이전 프롬프트 상태로 되돌아갑니다.

## ⚡ 터미널 내 Bash 명령어 실행 (`!`)
클로드 코드 안에서 깃 상태를 보거나 서버를 켤 때, 터미널을 새로 열 필요가 없습니다. 명령어 앞에 `!`를 붙이면 클로드 코드 내부에서 로컬 배시(Bash) 명령이 실행됩니다.
* `!git status` : 변경된 파일 확인
* `!npm run dev` : 로컬 서버 실행

## 💬 필수 슬래시 커맨드 (대화창 내부)
* `/init`: 프로젝트 설명서인 `CLAUDE.md`를 자동 생성합니다. (프로젝트 초기 세팅 시 필수)
* `/clear`: 쌓여있는 대화 컨텍스트를 완전히 초기화하여 토큰을 비웁니다.
* `/compact`: 대화 내역 중 **핵심 요약(Summary)**만 남기고 컨텍스트를 압축해 토큰을 절약합니다.
* `/context`: 현재 세션에서 사용 중인 토큰 양을 시각적인 바(Bar) 형태로 직관적으로 확인합니다.
* `/memory`: 현재 AI가 로드해서 읽고 있는 메모리(`CLAUDE.md` 등)의 내용을 확인합니다.
* `/status`: 현재 로그인된 계정, 구독 플랜, 연결된 MCP 서버 상태를 한눈에 종합적으로 확인합니다.
* `/allowed-tools`: 현재 AI에게 허용된 도구(파일 쓰기, 로컬 명령어 실행 등) 목록을 확인하고 권한을 추가하거나 삭제합니다.
* `/model`: 작업 난이도에 따라 모델을 전환해 토큰을 최적화합니다.
  * `Opus`: 복잡한 아키텍처 설계나 버그 해결 (가장 똑똑함)
  * `Sonnet`: 여러 에이전트에게 병렬로 일을 맡기거나 일반적인 코딩 작업
  * `Haiku`: 파일 탐색 등 간단하고 빠른 작업
* `/mcp`: 연결된 MCP 서버 목록을 확인합니다.
* `/export`: 현재 세션의 컨텍스트를 다른 챗봇(예: 웹 버전 클로드나 제미나이)으로 가져갈 수 있게 추출합니다.
* `/output-style`: AI의 응답 스타일을 변경합니다. '설명 모드(Explanatory)'나 '경고 모드(Warning)'로 설정하면 코딩 학습용으로 매우 좋습니다.

## 🛠️ 나만의 커스텀 슬래시 명령어 만들기
자주 쓰는 파이프라인을 아예 슬래시 명령어로 모듈화할 수 있습니다. 
1. 프로젝트 내 `.claude/commands/` 폴더를 생성합니다.
2. `ship.md` 등 원하는 이름으로 파일을 만들고 실행할 워크플로우(테스트 -> 커밋 -> 푸시)를 적습니다.
3. 클로드 코드 재시작 후 `/ship`을 입력하면 해당 파이프라인이 자동 실행됩니다!
