# 보이스 모드 사용법

Claude Code는 음성 입력 모드를 지원합니다. 타이핑 없이 말로 지시하고 싶을 때 유용한 기능입니다.

---

## 보이스 모드 활성화

터미널에서 `/voice` 명령어를 입력하면 음성 입력 모드가 활성화됩니다.

```bash
/voice
```

활성화되면 터미널 하단에 🎤 마이크 아이콘이 표시됩니다.

---

## 사용 방법

**기본 플로우:**

Space 키를 꾹 누른 상태에서 말을 시작합니다. 예를 들어 "유저 테이블에 이메일 컬럼 추가해줘"라고 말할 수 있습니다. 말이 끝나면 Space 키에서 손을 떼면 음성이 텍스트로 변환됩니다. 이때 자동으로 전송되지 않으므로, 변환 결과를 확인하고 필요하면 수정한 뒤 Enter 키를 눌러 전송합니다.

**주의할 점:**

손을 떼자마자 바로 전송되는 것이 아니라 텍스트로 변환된 후 직접 Enter를 눌러야 전송됩니다. 변환 결과를 확인하고 수정할 기회가 있다는 뜻입니다.

---

## 한국어 설정

기본값은 영어로 설정되어 있습니다. 한국어 음성 인식을 사용하려면 `/config` 명령어를 입력한 뒤 `Language` 항목에서 `Korean`을 선택하면 됩니다.

```bash
/config
```

### ⚠️ 한글 입력 충돌 이슈

한국어 IME(입력기)가 켜진 상태에서 `/config`에 `Korean`을 입력하면 Enter 키가 한글 확정으로 먼저 인식되어 설정이 제대로 적용되지 않는 문제가 있습니다.

**해결하는 방법:**

영어 입력 모드로 전환합니다 (Ctrl + Space 또는 Shift + Space). 그 상태에서 `/config`를 입력하고 Language 항목에 Korean을 입력한 뒤 Enter로 확정하면 정상적으로 설정됩니다.

---

## 비활성화

동일한 `/voice` 명령어를 다시 입력하면 토글 방식으로 비활성화됩니다.

```bash
/voice
```

---

## ✅ 좋은 사용 사례 vs ❌ 피해야 할 경우

**음성 모드가 적합한 상황:**

긴 자연어 지시가 필요할 때 유용합니다. 아키텍처 설명이나 요구사항 전달처럼 문맥이 긴 경우, 빠른 질문과 답변을 주고받을 때, 손을 키보드에서 떼고 생각하며 말하고 싶을 때, 화면을 보며 UI 변경을 지시할 때 적합합니다.

**음성 모드를 피해야 하는 상황:**

정확한 코드 스니펫을 입력해야 할 때는 직접 타이핑하는 것이 낫습니다. 파일명, 경로, API 키처럼 정확한 문자열이 필요한 경우, JSON이나 YAML 같은 구조화된 데이터를 입력할 때, 터미널 에러 로그를 복붙해야 할 때는 음성보다 직접 입력이 더 정확합니다.

**권장하는 하이브리드 사용법:**

음성으로 큰 그림을 지시한 뒤 디테일은 직접 타이핑으로 보정하는 방식이 가장 효율적입니다.

---

## 트러블슈팅

### 음성이 인식되지 않을 때

마이크 권한을 먼저 확인하세요 (OS 시스템 설정에서). Space 키를 충분히 오래 누르고 있는지 확인하고, `/voice`를 재입력해서 재활성화하거나 터미널을 재시작해보세요.

### 한국어가 영어로 인식될 때

`/config` 명령어로 들어가서 Language가 Korean으로 설정되어 있는지 확인하세요. 만약 English로 돌아가 있다면 위의 "한글 입력 충돌 이슈" 섹션을 참고해서 다시 설정하면 됩니다.

### 변환된 텍스트가 이상할 때

음성 인식 정확도는 발음, 주변 소음, 마이크 품질에 따라 달라집니다. 변환 후 Enter를 누르기 전에 반드시 확인하고 수정하세요. 특히 배포나 삭제 같은 중요한 작업은 직접 타이핑하는 것을 권장합니다.

---

## 💡 실전 활용 팁

### Plan Mode와 조합

보이스 모드로 "유저 프로필 편집 기능 만들어줘. 이름, 이메일, 프로필 사진 변경 가능하게"라고 말한 뒤 Shift + Tab으로 플랜을 확인합니다. 그다음 세부 수정은 타이핑으로 보정하는 방식입니다.

### 긴 요구사항 전달

복잡한 비즈니스 로직이나 여러 단계의 플로우는 말로 설명하는 게 오히려 빠를 수 있습니다. 예를 들어 "회원가입 플로우를 만들어줘. 이메일 중복 체크 → OTP 전송 → 인증번호 확인 → 비밀번호 설정 → 프로필 입력 순서로. Supabase Auth 사용하고, 각 단계마다 유효성 검사 넣어줘"처럼 한 번에 전달할 수 있습니다.

### 멀티태스킹

코드 리뷰를 하면서, 또는 다른 작업을 하는 중에 손을 떼고 Claude에게 지시할 수 있습니다. 키보드에서 손을 떼고 자유롭게 작업하면서 음성으로 명령을 내릴 수 있다는 점이 장점입니다.
