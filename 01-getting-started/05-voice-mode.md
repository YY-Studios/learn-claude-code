# 보이스 모드 사용법

Claude Code는 **음성 입력 모드**를 지원합니다. 타이핑 없이 말로 지시하고 싶을 때 유용합니다.

---

## 보이스 모드 활성화
```bash
/voice
```

활성화되면 터미널 하단에 🎤 마이크 아이콘이 표시됩니다.

---

## 사용 방법
```
1. Space 키를 꾹 누르기 (Press and Hold)
2. 말하기 (예: "유저 테이블에 이메일 컬럼 추가해줘")
3. Space 키에서 손 떼기 → 음성이 텍스트로 변환됨
4. Enter 키로 전송
```

**주의:** 손을 떼자마자 바로 전송되는 게 아니라, 텍스트로 변환된 후 **직접 Enter를 눌러야** 전송됩니다. 변환 결과를 확인하고 수정할 기회가 있습니다.

---

## 한국어 설정

기본값은 영어입니다. 한국어 음성 인식을 사용하려면:
```bash
/config
```

→ `Language` 항목에서 `Korean` 선택

### ⚠️ 한글 입력 충돌 이슈

한국어 IME(입력기)가 켜진 상태에서 `/config`에 `Korean`을 입력하면 Enter 키가 **한글 확정**으로 먼저 인식되어 설정이 제대로 적용되지 않습니다.

**해결 방법:**
```
1. 영어 입력 모드로 전환 (Ctrl + Space 또는 Shift + Space)
2. /config 입력
3. Language 항목에서 Korean 입력 (영문 모드 상태에서)
4. Enter로 확정
```

## 비활성화
```bash
/voice
```

동일한 명령어를 다시 입력하면 토글 방식으로 비활성화됩니다.

---

## ✅ 좋은 사용 사례 vs ❌ 피해야 할 경우

| ✅ 적합한 상황 | ❌ 부적합한 상황 |
|---|---|
| 긴 자연어 지시 (아키텍처 설명, 요구사항 전달) | 정확한 코드 스니펫 입력 |
| 빠른 질문 & 답변 | 파일명, 경로, API 키 등 정확한 문자열 |
| 손을 키보드에서 떼고 생각하며 말하기 | JSON, YAML 같은 구조화된 데이터 입력 |
| 화면을 보며 UI 변경 지시 | 터미널 에러 로그 복붙 |

**권장 하이브리드 사용법:**
```
음성으로 큰 그림 지시 → 디테일은 직접 타이핑으로 보정
```

## 트러블슈팅

### 음성이 인식되지 않아요
```
1. 마이크 권한 확인 (OS 시스템 설정)
2. Space 키를 충분히 오래 누르고 있는지 확인
3. /voice 재입력해서 재활성화
4. 터미널 재시작
```

### 한국어가 영어로 인식돼요
```
/config → Language가 Korean으로 설정됐는지 확인
만약 English로 돌아가 있으면 위 "한글 입력 충돌 이슈" 참고
```

### 변환된 텍스트가 이상해요
```
음성 인식 정확도는 발음, 주변 소음, 마이크 품질에 따라 달라집니다.
변환 후 Enter 전에 반드시 확인하고 수정하세요.
중요한 작업(배포, 삭제 명령)은 직접 타이핑 권장.
```

## 💡 실전 활용 팁

**Plan Mode와 조합**
```
1. 보이스 모드로 "유저 프로필 편집 기능 만들어줘. 이름, 이메일, 프로필 사진 변경 가능하게"
2. Shift + Tab으로 플랜 확인
3. 세부 수정은 타이핑으로 보정
```

**긴 요구사항 전달**

복잡한 비즈니스 로직이나 여러 단계의 플로우는 말로 설명하는 게 오히려 빠를 수 있습니다.
```
"회원가입 플로우를 만들어줘. 이메일 중복 체크 → OTP 전송 → 인증번호 확인 → 비밀번호 설정 → 프로필 입력 순서로. Supabase Auth 사용하고, 각 단계마다 유효성 검사 넣어줘."
```

**멀티태스킹**

코드 리뷰하면서, 또는 다른 작업 중에 손을 떼고 Claude에게 지시할 수 있습니다.
