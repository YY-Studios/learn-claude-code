# 자동 메모리 시스템 (/memory)

Claude Code는 반복되는 사용자의 작업 패턴을 학습하는 메모리 기능을 제공합니다.

* `/memory` 명령어를 입력하면 현재 Claude가 기억하고 있는 사용자의 선호도, 코딩 스타일, 프로젝트의 주요 특징을 확인할 수 있습니다.
* 이 데이터는 내부적으로 `MEMORY.md` 형태로 관리되며, 팀원과 공유해야 하는 공통 규칙(`CLAUDE.md`)과는 분리된 **개인화된 지식 저장소(Second Brain)** 역할을 합니다.
* AI가 같은 실수를 반복한다면, "이 프론트엔드 컨벤션을 메모리에 저장해 둬"라고 명시적으로 지시하여 다음 세션부터 자동으로 반영되도록 할 수 있습니다.
