# 효과적인 지시 패턴 (Think Hard 활용법)

클로드의 성능을 한계까지 끌어올리는 강력한 프롬프트 패턴들입니다.

## 🧠 깊은 추론 유도하기 (`think hard`)
클로드는 기본적으로 가장 빠르고 효율적인 답변을 내놓으려 합니다. 하지만 원인을 알 수 없는 복잡한 버그나 아키텍처 설계를 요구할 때는, 프롬프트 맨 앞에 **`think hard`** 또는 **`think harder`**를 붙여보세요. 

> "think hard. 현재 로그인 후 리다이렉트 과정에서 발생하는 무한 루프의 근본 원인을 스택 트레이스를 기반으로 깊게 분석하고, 3가지 해결 옵션을 장단점과 함께 제시해 줘."

이 마법의 단어를 붙이면 클로드가 즉시 코드를 내뱉기 전에, 내부적으로 여러 가지 시나리오를 검토하는 '사고 과정(Thinking)'을 훨씬 길고 밀도 있게 거쳐 압도적으로 높은 퀄리티의 결과물을 도출합니다.

## 🗣️ 음성 입력 (Whisper Flow)
복잡한 의도는 타이핑하다 지치기 마련입니다. 클로드 코드의 위스퍼 플로우 기능을 통해 말로 지시하세요. "이 버튼을 누르면 모달이 뜨는데, 트랜지션은 부드럽게 0.3초 딜레이를 주고..." 처럼 의식의 흐름대로 풍부하게 설명할수록 AI가 의도를 더 정확히 파악합니다.
