# 무거운 작업 스크립트 오프로드 (Offload)

대용량 JSON 파싱, 수백 개의 파일 포맷 일괄 변경, 복잡한 DB 데이터 추출 등을 Claude 대화창 내에서 직접 지시하면 컨텍스트 토큰이 순식간에 고갈되어 에러가 발생합니다.

## 💡 오프로드(Offload) 패턴
무거운 연산이나 반복 작업은 AI가 직접 수행하게 하지 말고, **처리를 위한 스크립트 작성만 지시**하세요.

* **Bad:** "이 `data.json` 파일 안의 1만 줄 데이터를 분석해서 카테고리별로 분류해 줘." (컨텍스트 폭발)
* **Good:** "이 `data.json`을 분석할 Node.js(또는 Python) 스크립트를 짜줘." -> 작성된 스크립트를 로컬 터미널에서 실행 -> **"실행 결과 요약본만 터미널에 출력됐는데, 이것을 바탕으로 다음 UI 작업을 진행해."**
