# TODO.md 주도 개발 (작업 연속성 유지)

여러 세션에 걸쳐 작업을 하다 보면 AI가 자신이 어디까지 개발했는지, 다음 목표가 무엇인지 맥락을 잃기 쉽습니다. 프로젝트 루트에 `TODO.md`를 만들어 AI가 스스로 작업 진척도를 관리하게 하세요.

## 🚀 실전 적용 워크플로우
1. 프로젝트 루트에 `TODO.md` 파일을 생성하고 체크리스트 뼈대를 만듭니다.
2. **세션 시작 시:** "TODO.md를 읽고 오늘 진행할 작업을 파악해서 Plan을 세워줘."
3. **작업 중:** 기능 하나가 끝날 때마다 "TODO.md에 체크 표시해 줘."
4. **세션 종료 전:** "오늘 완료한 작업과 다음에 이어서 할 작업을 TODO.md에 자세히 업데이트하고 세션을 종료할게."

이 방식을 통해 토큰을 비우기 위해 `/clear`를 하더라도, 다음 세션에서 `TODO.md` 하나만 읽게 하면 완벽하게 작업 연속성을 이어갈 수 있습니다.
