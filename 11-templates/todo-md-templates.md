## CLAUDE.md에 넣는 TODO 읽기 규칙

```markdown
## 작업 규칙
- 세션 시작 시 항상 TODO.md를 읽고 현재 상태 파악
- 기능 완료 시마다 TODO.md 업데이트 (완료 항목 체크, 다음 예정 추가)
- 중요한 기술 결정은 의사결정 메모 섹션에 날짜와 함께 기록
- 세션 종료 전 오늘 완료한 내용과 다음 작업을 TODO.md에 정리 후 커밋
```

---

## 세션별 사용 예시

**세션 시작 시 프롬프트:**
```
TODO.md를 읽고 오늘 진행할 작업을 파악한 뒤 Plan을 세워줘.
```

**작업 완료 시 프롬프트:**
```
방금 완료한 [기능명]을 TODO.md에 체크하고, 다음 예정 항목도 업데이트해줘.
```

**세션 종료 전 프롬프트:**
```
오늘 완료한 작업, 다음에 이어할 작업, 중요한 결정 사항을
TODO.md에 정리하고 git commit해줘.
```
