# 커스텀 슬래시 명령어 만들기 (.claude/commands)

매번 반복하는 테스트, 포맷팅, 깃 커밋 과정을 아예 나만의 슬래시 명령어(Slash Command)로 모듈화하여 에이전트 파이프라인을 구축할 수 있습니다.

## 🚀 생성 방법
1. 프로젝트 루트 디렉토리에 `.claude/commands/` 폴더를 만듭니다.
2. 실행할 명령어 이름으로 마크다운 파일을 생성합니다. (예: `ship.md`)
3. 파일 내부에 실행할 워크플로우를 프롬프트 형태로 작성합니다.

**`ship.md` 작성 예시:**
```md
# Description
테스트 실행 후 성공 시 커밋하고 푸시합니다.

# Workflow
1. `npm run test`를 실행한다.
2. 테스트 실패 시 에러 로그를 출력하고 즉시 작업을 중단한다.
3. 테스트 통과 시 `git add .`를 실행한다.
4. 변경 사항을 분석하여 알맞은 커밋 메시지로 `git commit`을 진행한다.
5. `git push`를 실행하여 원격 저장소에 배포한다.
