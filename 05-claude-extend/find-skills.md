# 스킬 찾기 & 설치

직접 만들지 않고 커뮤니티에서 만든 스킬을 가져다 쓸 수 있습니다.

---

## 스킬을 찾을 수 있는 곳

| 플랫폼 | 특징 |
|---|---|
| **Anthropic 공식 마켓플레이스** | Claude Code 내 `/plugins` 명령어로 접속 |
| **Skills.sh** | 설치 수 기반 인기 리더보드 제공 |
| **Skills.mcp** | 카테고리별 정리로 분야별 탐색에 유리 |
| **Skill.bu** | 브라우저에서 미리 써볼 수 있는 플레이그라운드 (준비 중) |

---

## 추천 스킬

### Find Skills (처음이라면 필수)
스킬을 찾아주는 스킬입니다. 자연어로 요청하면 관련 스킬과 설치 명령어를 안내해 줍니다.
```
"카드 뉴스 만드는 스킬 찾아줘"
```

### Commit Command
코드 수정 후 커밋 메시지를 자동 작성해 주는 플러그인입니다.

### Skill Creator
나만의 맞춤형 스킬을 만들어주는 도구입니다. → [스킬 직접 만들기](./skills.md) 참고

---

## 설치 방법

Claude Code 내 `/plugins` 명령어로 마켓플레이스에 접속하거나, 터미널에서 직접 설치할 수 있습니다.

---

## ⚠️ 주의사항

- **설치 개수 제한**: 너무 많으면 AI가 일부 스킬을 인식하지 못할 수 있습니다. **40개 내외**로 유지하세요.
- **보안 확인**: 외부 스킬은 파일 삭제나 외부 호출을 포함할 수 있습니다. 설치 전 GitHub에서 마크다운 파일을 확인하세요.
